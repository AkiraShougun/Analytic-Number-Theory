The Mobius Function

Number theory is often concerned with sequences of real or complex numbers. We call such sequences, arithmetical functions.

Definition

The mobius function \(\mu\) is defined as follows: \[\mu(1)=1;\] If \(n>1\), write \(n = p_1^{a_1}\cdots p_k^{a_k}\).Then \[\mu(n)=(-1)^k \text{ if } a_1=\cdots=a_k=1\] \[\mu(n)=0 \text{ otherwise.}\]

Notice that \(\mu(n) = 0 \) if and only if n has a square factor > 1.

Until now, we only have introduced the Mobius function and now you might wonder, "what's the point of using such a function? Why is it used in analytic number theory?"
Well, from now on we will introduce theorems regarding this function that will help us eventually solve more complex problems very easily.

Theorem

\(\forall n \ge 1,\) \[\sum_{d|n}\mu(d)= \left\lfloor \frac{1}{n} \right\rfloor = \left\{ \begin{array}{cl} 1 & : \ n = 1. \\ 0 & : \ n > 1. \end{array} \right. \]

Proof.

It is trivial to see that the formula is clearly true if n=1.
Assume, then, that n>1 and write, by the fundamental theorem of arithmetic, \[n=p_1^{a_1} \cdots p_k^{a_k}\] In the sum \(\sum_{d|n}Î¼(d)\) the only non-zero terms come from d=1 and the divisors of n which are products of distinct primes. Thus, \[\sum_{d|n}\mu(d)=\mu(1)+\mu(p_1)+\cdots+\mu(p_k)+\mu(p_1p_2)+\cdots+\mu(p_{k-1}p_k)+\cdots+\mu(p_1p_2\cdots p_k)\\ =1+\binom{k}{1}(-1)+\binom{k}{1}(-1)^2+ \cdots+\binom{k}{k}(-1)^k=(1-1)^k=0 \text{. } \square \]

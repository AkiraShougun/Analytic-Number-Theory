Definition

\(\forall n \ge 1\), the Euler totient \(\phi(n)\) is defined as follows, \[\phi(n)=|\left\{ {m∈N:m≤n,(m,n)=1} \right\}|\]

Remember our previous sum \(\sum_{d|n}\mu(d)\) for \(\mu(n)\)? Now we are going to introduce a similar sum involving the newly introduced function.

Theorem

\(\forall n \ge 1\), we have, \[\sum_{d|n}\phi(d)=n\]

Proof

Let \(\Sigma\) denote the set \(\left\{1,2,...,n\right\}\). We distribute the integers of \(\Sigma\) into disjoint sets as the following:
For each divisor \(d\) of \(n\), let \[A(d)= |\left\{ k:(k,n) = d,1 \le k \le n \right\}|.\] So if f(d) denotes the number of integers in \(A(d)\) we have \[\sum_{d|n}f(d)=n\] However \((k,n)\)=1 if and only if \((k/d,n/d)=1\), and \(1 \lt k \le n\) if and only if \(0 \lt k/d \le n/d\). Therefore, it is very obvious to see that if we let \(q=k/d\), there is a one-to-one correspondence between the elements in \(A(d)\) and those integers \(q\) satisfying \(0 \lt q \le n/d\), \((q,n/d)=1\). The number of such \(q\) is \(\phi(n/d)\). Hence \(f(d)=\phi(n/d)\) and we get, \[\sum_{d|n}\phi(n/d)=n\] But this statement is enough to prove our theorem. \(\square\)
